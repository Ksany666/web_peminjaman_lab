# 🚀 Sistem Peminjaman Laboratorium UMMI

🔥 Modern • Responsive • Easy to Use

Website Sistem Peminjaman Laboratorium berbasis PHP Native & MySQL yang dirancang untuk mempermudah proses pengajuan, persetujuan, dan pengelolaan peminjaman laboratorium secara digital.

## ✨ Fitur Unggulan
- 🔐 Login Multi User
- 👤 Manajemen User
- 🏢 Manajemen Laboratorium
- 📝 Peminjaman Laboratorium
- ✅ Approve & Reject Pengajuan
- 📎 Upload Dokumen Pendukung
- ✍️ Digital Signature (Tanda Tangan)
- 📊 DataTables (Search, Sort & Pagination)
- 📄 Export Excel
- 📱 Responsive Bootstrap 5

## 🛠️ Tech Stack
- PHP Native
- MySQL
- Bootstrap 5
- JavaScript
- jQuery
- DataTables
